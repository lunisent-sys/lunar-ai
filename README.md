# 🌙 Lunar AI - Intelligent Chat Assistant

Lunar AI adalah asisten pintar berbasis web dengan tampilan glassmorphism yang elegan, didukung oleh Groq AI untuk respons yang cepat dan akurat.

## ✨ Fitur Utama

- **Glassmorphism UI** - Tampilan modern dengan efek kaca transparan
- **Sistem Login/Register** - Multi-user dengan penyimpanan lokal
- **Memori per User** - AI mengingat percakapan setiap user
- **Upload File** - Support gambar, PDF, dokumen (max 10MB)
- **Responsive Design** - Bekerja sempurna di semua ukuran layar
- **Chat History** - Riwayat percakapan tersimpan
